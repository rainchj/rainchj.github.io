1. Market Monitor Pro 기능 및 특징




  • 환율시장, 채권시장, 주식시장의 주요지표와 관심종목으로 등록한 개별종목을 1분 단위로 체크하여 전일비, 고가/저가, Z-score, 분당 증감율, 거래량 등에 이상징후가 발생한 경우 즉시 사용자에게 점검결과를 보여줍니다.
  • 또한 환율, 국채금리, 알림 체크 종목에 대하여는 소리, 팝업창, 텔레그램 전송 등을 통해 사용자가 실시간으로 이상징후를 인지할 수 있도록 도와줍니다.
  • 텔레그램 전송 시에는 전달받은 메세지의 해당 종목을 클릭하면 네이버증권 해당종목의 상세 페이지로 바로 이동합니다.
    • 텔레그램 전송 문자 예시
    [Market Monitor Pro 05.10/13:44]
    • KOSPI: 36월 고가
    • 삼성전자: 전일 +3.1%
    • KODEX 200: Zscore +2.0

  • 국내에 상장된 모든 개별종목과 지수(KOSPI, KOSDAQ 등)를 지원하며, 주식시장은 장이 열리는 09:00 ~ 15:30 동안 작동하고 이외의 시간에는 점검, 알림이 중지됩니다. 환율, 금리 등 중요 시장지표는 상시 점검하고 알림이 발송됩니다.



2. 점검항목 및 알림기준



2.1 상시 점검항목 및 알림기준



2.1.1 환율 증감율

  • 알림 발생기준 : 1.0% 이상 상승(또는 하락)
    팝업, 텔레그램 전송 : 0.5% (예시: 1.0%, 1.5%, 2.0% 등) 단위로 최초 도래시

2.1.2 국채금리 증감율

  • 알림 발생기준 : 1.0% 이상 상승(또는 하락)
    팝업, 텔레그램 전송 : 0.5% (예시: 1.0%, 1.5%, 2.0% 등) 단위로 최초 도래시


2.1 장중(09:00~15:30) 점검항목 및 알림 기준



2.1.1 전일비

  • 알림 발생기준 : 2.0% 이상 상승(또는 하락)
    팝업, 텔레그램 전송 : 0.5% (예시: 2.0%, 2.5%, 3.0% 등) 단위로 최초 도래시

2.1.2 기간중 최고가, 최저가

  • 알림 발생기준 : 12개월 동안의 최고가(또는 최저가) 이상 상승(또는 하락)
    팝업, 텔레그램 전송 : 최고가(최저가) 도달시 1회 전송

2.1.3 Z-score (Bolinger Band와 동일)

  • 알림 발생기준 : 2.0 이상 상승(또는 하락)
    팝업, 텔레그램 전송 : 0.2 (예시: 2.0, 2.2, 2.4 등) 단위로 최초 도래시
  • 이동평균기간 설정값 : 20일
    10일: 단기, 20일 : 표준, 60일: 중기, 120일: 장기

2.1.4 분당 증감율

  • 알림 발생기준 : 0.4% 이상 상승(또는 하락)
    팝업, 텔레그램 전송 : 0.2% (예시: 0.4%, 0.6%, 0.8% 등) 단위로 최초 도래시

2.1.5 거래량 증가

  • 알림 발생기준 : 2.0배 이상 상승(또는 하락)
    팝업, 텔레그램 전송 : 0.5배 (예시: 2.0배, 2.5배, 3.0배 등) 단위로 최초 도래시

* 모든 점검항목의 설정값은 '설정' 메뉴에서 수정후 저장하면 변경할 수 있습니다.



3. 관심종목 추가 및 삭제


3.1 '종목편집' 버튼을 통한 관심종목 추가 및 삭제

  • 종목명, 종목코드를 제공되는 목록에서 선택하여 추가 및 삭제할 수 있습니다.
    제공되는 목록은 코스피는 시총 상위 300개 기업, ETF는 시총 2,000억원 이상 종목입니다.

3.2 수기로 관심종목 추가 및 삭제

  • 제공 목록에 없는 종목은 수기로 추가할 수 있습니다.
    프로그램이 설치된 폴더에 'settings.json' 파일을 에디터로 불러옵니다.
    이 파일의 "indices" 항목에 종목명과 종목코드를 추가, 삭제 후 저장합니다.
    관심종목의 코드는 네이버 금융에서 찾을 수 있습니다.
[settings.json]

"indices": {
"KOSPI":"KOSPI",
"KOSDAQ":"KOSDAQ",
"삼성전자":"005930",
"추가할 종목명":"종목코드",
.
.
}




4. 텔레그램 전송 설정방법


  • 모바일에서 텔레그램 앱을 설치하고 계정가입 후에 아래를 참조하여 텔레그램 봇 생성, BOT TOKEN, CHAT ID를 발급받습니다.
  • 프로그램의 '설정'에서 '텔레그램 알림'을 체크하고 발급받은 BOT TOKEN, CHAT ID를 입력한 후 저장합니다.
  • 'settings.json' 파일에서 수기로 입력할 수도 있습니다.
[ 텔레그램 봇 생성, BOT TOKEN 및 CHAT ID 발급 방법 ]

1. 모바일에서 텔레그램 설치 및 로그인

2. 텔레그램 검색창에서 "BotFather" 검색 후 공식계정 선택

3. BotFather 채팅창에서 아래 명령 입력
- /newbot

4. 순서대로 입력
- BOT 이름 입력
- BOT 아이디 입력 (반드시 bot 으로 끝나야 함)
- 예시: stock_alarm_bot

5. 생성 완료 후 발급되는 BOT TOKEN 복사
- 예시: 123456789:AAxxxxxxxxxxxxxxxx

6. 생성된 BOT 채팅방에 들어가 아래 명령 입력
- /start

7. 브라우저 주소창에 아래 입력하여 CHAT ID 확인
- https://api.telegram.org/bot여기에토큰/getUpdates

8. 결과 JSON 의 "id" 숫자가 CHAT ID
- 예시: "id": 123456789
[ settings.json 수기 수정 방법]

"telegram_enabled": true,
"telegram_token": "YOUR_TELEGRAM_BOT_TOKEN",
"telegram_chat_id": "YOUR_TELEGRAM_CHAT_ID"

  • 윈도우가 절전모드로 진입하면 프로그램이 동작하지 않습니다.
    '제어판 > 전원옵션 > 전원관리 옵션설정 편집 > 컴퓨터를 절전모드로 설정 > 해당없음'으로 설정하면 절전모드 진입이 차단되어 알림을 계속 받을 수 있습니다.



5. 버전별 주요내용


5-1 Market Monitor Pro 1.04

  • 관심종목을 제공된 목록에서 선택 및 해제할 수 있도록 개선
    코스피는 시총 상위 100개, ETF는 시총 5,000억원 이상인 종목들 목록 제공
  • 고가/저가 기간 설정 추가
    관심종목에 대해서 설정 기간에 대한 최고가, 최저가 점검 기능 추가
  • 동일 사유로 발생한 알림은 1회로 제한하여 팝업/텔레그램 알림 횟수 최소화

5-2 Market Monitor Pro 1.05

  • 치명적인 오류 수정
    종목편집 버튼을 통한 관심종목의 추가/삭제/위치이동 시, 설정 버튼을 통한 고가/저가 기간 변경 시 DATA 혼선 발생: 해당 작업후 프로그램을 리셋하여 DATA 혼선 방지
  • 불편사항 개선 및 소소한 오류 수정
    종목 추가시 코스피는 시총 상위 300개 기업, ETF는 시총 4,000억원 이상인 종목들 목록 제공
    15:30분 장마감후 최종 가격 미반영 방지를 위해 15:31분까지 점검, 알림 시간을 연장
    알림기준 기본값 변경, 컬럼길이 확대 등

5-3 Market Monitor Pro 1.06

  • 거래량 증가를 점검 및 알림항목에 신규 추가

    전일대비 1.0%('설정'에서 변경 가능) 이상 상승(+) 또는 하락(-)
