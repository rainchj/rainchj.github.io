1. Market Monitor Pro 기능 및 특징




  • 관심종목으로 등록한 주식을 1분 단위로 실시간 체크하여 이상징후를 점검하고 알림 체크가 되어있는 종목에 한하여 소리, 팝업창, 텔레그램으로 알림 정보를 제공합니다.
  • 텔레그램으로 알림이 전달되어 모바일에서도 실시간 확인할 수 있으며 메세지의 종목을 터치하면 네이버증권 해당 종목의 상세페이지로 바로 이동합니다.
    • [Market Monitor 26.05.10 13:44]
    • KOSPI: 36월 고가
    • 삼성전자: 전일 +3.1%
    • KODEX 200: Zscore +2.0

  • 국내에 상장된 모든 개별종목과 지수(KOSPI, KOSDAQ 등)를 지원하며, 장이 열리는 09:00 ~ 15:35 동안 작동하고 이외의 시간에는 점검, 알림이 중지됩니다.



2. 이상징후 점검항목 및 알림 기준


2.1 전일비

  • 알림 기본 설정값 : 3.0% 이상 상승(+) 또는 하락(-)
    수정하려면 '설정'의 '전일비 알림기준'에 원하는 %값을 +, -는 제거하고 입력합니다.
    종목별 개별 설정시에는 메인화면의 '종목전일비' 입력박스에 입력합니다.

2.2 기간중 최고가, 최저가

  • 알림 기본 설정값 : 36개월 동안 최고가 또는 최저가
    수정하려면 '설정'의 '고가/저가 기간'에 원하는 기간을 개월 단위로 입력합니다.
    개월수가 늘어나면 프로그램 시작시 데이터를 불러오는 시간이 길어집니다.

2.3 Z-score (Bolinger Band와 동일)

  • 알림 기본 설정값 : 2.2 이상 상승(+) 또는 하락(-)
    수정하려면 '설정'의 'Z-score 알림기준'에 원하는 값을 +, -는 제거하고 입력합니다.
    * Z-score(ma 20 기준): +1 강, +2 과열, +3 극단과열, -1 약, -2 과매도, -3 패닉
  • 이동평균 기간 기본 설정값 : 20일
    10일: 단기, 20일 : 표준, 60일: 중기, 120일: 장기

2.4 분당 증감율

  • 알림 기본 설정값 : 0.5% 이상 상승(+) 또는 하락(-)
    수정하려면 '설정'의 '분증가율 알림기준'에 원하는 %값을 +, -는 제거하고 입력합니다.



3. 관심종목 추가 및 삭제


3.1 '종목편집' 버튼을 통한 관심종목 추가 및 삭제

  • 종목명, 종목코드를 제공되는 목록에서 선택하여 추가 및 삭제할 수 있습니다.
    제공되는 목록은 코스피는 시총 상위 300개 기업, ETF는 시총 4,000억원 이상 종목입니다.

3.2 수기로 관심종목 추가 및 삭제

  • 제공 목록에 없는 종목은 수기로 추가, 삭제할 수 있습니다.
    프로그램 설치 후 처음 실행하면 프로그램이 설치된 폴더 내에 'settings.json' 파일이 자동 생성됩니다.
    이 파일의 "indices" 항목에 종목명과 종목코드를 추가, 삭제한 후 프로그램을 실행하면 정상적으로 반영됩니다.
    관심종목의 코드는 네이버 금융에서 검색하여 추가하면 됩니다.
[settings.json]

"indices": {
"KOSPI":"KOSPI",
"KOSDAQ":"KOSDAQ",
"삼성전자":"005930",
"추가할 종목명":"종목코드",
.
.
}




4. 텔레그램 알림 설정


  • Market Monitor Pro가 주식 이상징후를 감지하면 텔레그램을 통해 모바일에서도 실시간으로 확인할 수 있습니다.
    모바일에서 텔레그램 앱을 설치하고 계정가입 후에 텔레그램 봇 생성, BOT TOKEN, CHAT ID를 발급받습니다.
    Market Monitor Pro '설정'에서 '텔레그램 알림을 체크'하고 발급받은 BOT TOKEN, CHAT ID 입력 후 저장합니다.
    'settings.json' 파일에서 직접 입력할 수도 있습니다.
[ 텔레그램 봇 생성, BOT TOKEN 및 CHAT ID 발급 방법 ]

1. 모바일에서 텔레그램 설치 및 로그인

2. 텔레그램 검색창에서 "BotFather" 검색 후 공식계정 선택

3. BotFather 채팅창에서 아래 명령 입력
- /newbot

4. 순서대로 입력
- BOT 이름 입력
- BOT 아이디 입력 (반드시 bot 으로 끝나야 함)
- 예시: stock_alarm_bot

5. 생성 완료 후 발급되는 BOT TOKEN 복사
- 예시: 123456789:AAxxxxxxxxxxxxxxxx

6. 생성된 BOT 채팅방에 들어가 아래 명령 입력
- /start

7. 브라우저 주소창에 아래 입력하여 CHAT ID 확인
- https://api.telegram.org/bot여기에토큰/getUpdates

8. 결과 JSON 의 "id" 숫자가 CHAT ID
- 예시: "id": 123456789
[ settings.json 수기 수정 방법]

"telegram_enabled": true,
"telegram_token": "YOUR_TELEGRAM_BOT_TOKEN",
"telegram_chat_id": "YOUR_TELEGRAM_CHAT_ID"
  • 윈도우가 절전모드로 진입하면 프로그램이 동작하지 않습니다.
    '제어판 > 전원옵션 > 전원관리 옵션설정 편집 > 컴퓨터를 절전모드로 설정 > 해당없음'으로 설정하면 절전모드 진입이 차단되어 알림을 계속 받을 수 있습니다.



5. 버전별 주요내용


5-1 Market Monitor Pro 1.04

  • 관심종목을 제공된 목록에서 선택 및 해제할 수 있도록 개선
    코스피는 시총 상위 100개, ETF는 시총 5,000억원 이상인 종목들 목록 제공
  • 고가/저가 기간 설정 추가
    관심종목에 대해서 설정 기간에 대한 최고가, 최저가 점검 기능 추가
  • 동일 사유로 발생한 알림은 1회로 제한하여 팝업/텔레그램 알림 횟수 최소화

5-2 Market Monitor Pro 1.05

  • 치명적인 오류 수정
    종목편집 버튼을 통한 관심종목의 추가/삭제/위치이동 시, 설정 버튼을 통한 고가/저가 기간 변경 시 DATA 혼선 발생: 해당 작업후 프로그램을 리셋하여 DATA 혼선 방지
  • 불편사항 개선 및 소소한 오류 수정
    종목 추가시 코스피는 시총 상위 300개 기업, ETF는 시총 4,000억원 이상인 종목들 목록 제공
    15:30분 장마감후 최종 가격 미반영 방지를 위해 15:35분까지 점검, 알림 시간을 연장
    알림기준 기본값 변경, 컬럼길이 확대 등
