지표별 과매도 · 과매수 자동 감지
아래 임계값을 넘으면 차트에 자동으로 신호를 표시하고, 신호 발생 횟수와 기간별 전환 확률을 함께 출력합니다.

지표 과매도 임계값 과매수 임계값
볼린저밴드 종가 ≤ BB Lower 종가 ≥ BB Upper
MACD MACD < 0 & 하향 MACD > 0 & 상향
RSI RSI ≤ 30 RSI ≥ 70
Slow STC STC K ≤ 20 STC K ≥ 80
Z-score Z-score ≤ −2.0 Z-score ≥ +2.0

시그널 신호

  • 과매수/과매도: BB, RSI, Slow STC, Z-score의 개별지표 or 복합지표
  • BB_기준선터치: BB 상하단밴드선이 동시 하락하다 하단밴드선이 수평되는 시점
  • BB_폭락주의: 상하단 밴드폭이 벌어지면서 종가가 하단밴드선을 타고 내려오는 시점
  • BB_폭등주의: 상하단 밴드폭이 벌어지면서 종가가 상단밴드선을 타고 올라가는 시점
  • SMA_정배열: SMA장기 > SMA중기 > SMA단기
  • SMA_역배열: SMA장기 < SMA중기 < SMA단기
  • MACD_골든크로스: MACD선이 Signal선을 밑에서 위로 교차하는 시점
  • MACD_데드크로스: MACD선이 Signal선을 위에서 아래로 교차하는 시점
  • MACD_강상승추세: MACD > 0 & 상승 + 히스토그램 > 0 & 상승
  • MACD_강하락추세: MACD < 0 & 하락 + 히스토그램 < 0 & 하락
  • MACD_상승추세약화: MACD > 0 & 하락 + 히스토그램 > 0 & 하락
  • MACD_하락추세약화: MACD < 0 & 상승 + 히스토그램 < 0 & 상승
  • 복합_매수신호1: BB 하단밴드 기울기 상승 + MACD Histogram 2일 연속 증가 + RSI < 40 & 상승 + STC < 20 & 상승

중복 신호 처리 — 두 가지 방식 동시 적용

제거 방식: 연속으로 신호가 발생한 경우 첫 신호 발생일만 분석 대상에 포함.
포함 방식: 연속 신호를 포함한 모든 신호 발생일을 분석 대상에 포함.
두 방식을 비교하면 신호의 독립성·신뢰도를 함께 평가할 수 있습니다.

📐

전환 판별 기준

상승 전환(과매도): 분석 기간 내 종가가 신호 발생일 종가보다 높은 날이 단 하루라도 존재하면 상승으로 판정.
하락 전환(과매수): 분석 기간 내 종가가 신호 발생일 종가보다 낮은 날이 단 하루라도 존재하면 하락으로 판정.

🧮

개별 + 복합 지표 분석

BB·MACD·RSI·STC·Z-score 각각의 단독 신호뿐 아니라, 여러 지표가 동시에 과매도·과매수를 가리키는 복합 신호도 별도로 분류해 확률을 산출합니다.

4

AI 분석 실행

AI 분석 버튼을 클릭하면 아래 보고서를 브라우저로 확인할 수 있습니다.

① 기술적 지표 분석 — 시계열 CSV → AI 전송 → [시장 상황 / 지표 분석 / 매수 전략] 리포트 생성
② 과매도 반전 확률 분석 — 과매도 시그널 JSON → AI 전송 → 기간별 상승 전환 확률 보고서 생성
③ 과매수 반전 확률 분석 — 과매수 시그널 JSON → AI 전송 → 기간별 하락 전환 확률 보고서 생성

소요 시간은 분석 기간, AI 모델, PC 성능 등에 따라 약 10초~3분 입니다.

[ 자주 묻는 질문 ]
궁금한 점을 확인하세요
과매도·과매수 시그널 분석은 어떻게 활용하나요? +
신호 발생일 이후 실제로 주가가 반전한 비율(빈도·확률)을 기간별로 보여줍니다. 예를 들어 "RSI 30 이하에서 10 거래일 내 상승 전환 확률이 85%였다"는 식의 과거 통계를 확인해, 현재 신호의 신뢰도를 스스로 판단하는 데 활용할 수 있습니다. 단, 과거 확률이 미래를 보장하지는 않으므로 참고 지표로만 사용하세요.
중복 신호 제거 방식과 포함 방식의 차이는 무엇인가요? +
제거 방식은 연속으로 신호가 발생한 경우 첫 번째 신호 날만 분석에 포함합니다. 독립적인 과매도·과매수 구간을 기준으로 통계를 내므로 중복 계산을 방지하고 신호의 순수한 신뢰도를 측정할 때 적합합니다.

포함 방식은 연속 신호를 포함해 모든 신호 발생일을 분석에 포함합니다. 신호 발생 빈도가 높아 통계 표본이 늘어나지만, 연속 구간이 결과에 더 많이 반영됩니다. 두 방식을 비교해 결과가 일치하면 신뢰도가 높은 신호로 볼 수 있습니다.
AI 분석이 3회 실행된다면 API 요청도 3배 소모되나요? +
맞습니다. 기술적 지표 분석(①), 과매도 반전 확률(②), 과매수 반전 확률(③) 각각 별도 API 요청이 발생합니다. Gemini 무료 플랜 기준 하루 20회(일부 모델) 한도이므로, 하루에 약 6~7회 분석이 가능합니다. 한도 초과 시 다른 API 키를 새로 발급해 사용하거나, 일일 한도가 높은 경량 모델(gemini-flash-lite 등)을 선택하세요.
엑셀 버전 제한이 있나요? +
Windows Excel 2016 이상을 권장합니다. VBA 매크로와 WinHttp, ADODB.Stream, MSXML2 등 Windows COM 객체를 사용하므로 Mac 환경에서는 정상 동작하지 않습니다.
압축 해제 비밀번호는 어디서 얻나요? +
본 프로그램은 지인 대상 제한 배포입니다. 네이버 블로그(U와 함께하는 행복한 세상살이) 또는 유튜브 채널(백점경제학) 댓글로 문의하시면 안내해 드립니다.
API 호출 비용이 많이 드나요? +
Gemini 무료 플랜 사용 시 비용이 없습니다(하루 500회 한도). 유료 모델은 분석 1회당 약 $0.001~$0.005 수준으로, 하루 10회 분석 기준 월 $1~2 내외입니다.
분석 결과를 투자에 바로 활용해도 되나요? +
AI 분석은 기술적 지표를 해석한 참고 자료입니다. 투자 결정은 반드시 본인의 판단 하에 이루어져야 하며, 분석 결과에 따른 투자 손익의 책임은 투자자 본인에게 있습니다.
해외 종목도 분석할 수 있나요? +
네. NASDAQ, S&P500, 개별 미국 주식 등 야후파이낸스에서 시세를 제공하는 종목이라면 대부분 분석 가능합니다. 종목 코드(티커)를 목록에 추가하면 됩니다.
API 키가 외부에 노출될 위험은 없나요? +
API 키는 엑셀 시트 셀에만 저장되며, AI 서버로의 HTTPS 통신에만 사용됩니다. 단, 엑셀 파일을 타인과 공유할 때는 API 키를 삭제한 후 공유하세요.
